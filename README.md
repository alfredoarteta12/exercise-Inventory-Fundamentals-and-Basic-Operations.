# exercise-Inventory-Fundamentals-and-Basic-Operations.en ingles

## User Story

## Simple Sales Registration


_As a user or seller, I want to register the product name, product quantity, and product price in a program
So that I can calculate and keep track of the total of each sale quickly and easily._

## Description:

_The system will allow the user to enter the basic data of a sale: product name, product quantity, and product price. With this information, the program will automatically calculate the total purchase and display the result to the user._

## Acceptance Criteria:

1. The user can enter the product name.

2. The user can enter the product quantity.

3. The user can enter the product price.

4. The system automatically calculates the total sale (price × quantity).

5. The system displays the total purchase to the user.
